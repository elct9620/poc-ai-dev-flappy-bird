# Events

The document provides an overview of the events used in the game engine and their definitions.

## GameEvent

Events generated by player actions or game mechanics.

| Name                                                       | Description                                          |
|------------------------------------------------------------|------------------------------------------------------|
| [CREATE_SCORE](./design/event/create_score.md)             | Creates a new score entity with full configuration   |
| [RESET_SCORE](./design/event/reset_score.md)               | Resets score value to initial state                  |
| [INCREMENT_SCORE](./design/event/increment_score.md)       | Increments score value by 1                          |
| [REMOVE_SCORE](./design/event/remove_score.md)             | Removes a score entity                               |
| [CREATE_BIRD](./design/event/create_bird.md)               | Creates a new bird entity with physics properties    |
| [BIRD_FLAP](./design/event/bird_flap.md)                   | Triggers bird flap action                            |
| [KILL_BIRD](./design/event/kill_bird.md)                   | Marks a bird as dead (sets isAlive to false)         |
| [REMOVE_BIRD](./design/event/remove_bird.md)               | Removes a bird entity                                |

## SystemEvent

Events generated by adapters or external systems.

| Name                                           | Description                           |
|------------------------------------------------|---------------------------------------|
| [MOUSE_CLICK](./design/event/mouse_click.md)   | Player mouse click on canvas          |
| [KEY_DOWN](./design/event/key_down.md)         | Player keyboard key press             |
| [TICK](./design/event/tick.md)                 | Main game loop frame update           |
