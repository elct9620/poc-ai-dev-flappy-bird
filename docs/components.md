# Components

This document provides an overview of the components used in the game engine and their definitions.

| Name                                           | Description                                                     |
|------------------------------------------------|-----------------------------------------------------------------|
| [Background](./design/component/background.md) | The visual representation of the tiled background scene.        |
| [Bird](./design/component/bird.md)             | The visual representation of the player-controlled bird.        |
| [Ground](./design/component/ground.md)         | The visual representation of the ground at the bottom of screen.|
| [Score](./design/component/score.md)           | The UI widget responsible for displaying the current score.    |
