# Renderers

This document provides an overview of the renderers used in the game engine and their definitions.

| Name                                           | Description                                                     |
|------------------------------------------------|-----------------------------------------------------------------|
| [Background](./design/renderer/background.md) | The visual representation of the tiled background scene.        |
| [Bird](./design/renderer/bird.md)             | The visual representation of the player-controlled bird.        |
| [Ground](./design/renderer/ground.md)         | The visual representation of the ground at the bottom of screen.|
| [Score](./design/renderer/score.md)           | The UI widget responsible for displaying the current score.    |
